# Sosa Barber Collective

Landing page estatica para una barberia premium en Bogota, inspirada en las referencias visuales del proyecto.

## Archivos principales

- `index.html`: estructura general de la landing.
- `styles.css`: diseno visual, responsive y estilo editorial.
- `content.js`: textos, precios, productos, redes, barberos, galeria y videos embebidos.

## Como actualizar contenido

Edita `content.js` para cambiar:

- nombres y textos de portada
- enlaces de WhatsApp y redes sociales
- precios y servicios
- nombres de los tres barberos
- productos y descripciones
- fotos usadas en galeria o tarjetas
- videos de YouTube, TikTok o Instagram mediante URLs de embed
- direccion, telefono, horario y mapa

## Notas

- El proyecto usa las imagenes y el logo que ya estaban en la carpeta.
- Si luego quieres convertirlo a React o agregar formulario real de reservas, se puede hacer sobre esta base.
